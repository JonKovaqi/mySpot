# mySpot

Linku per website : https://my-spot-sooty.vercel.app/index.html
mySpot është një platformë për rezervimin e akomodimeve, e ngjashme me Airbnb. Ndërtuar me HTML, CSS dhe JavaScript.

## Karakteristikat

- **Faqja Kryesore** - Hero section me karusel apartamentesh, seksion për pritës dhe recensione
- **Vendet** - Kërko apartamente sipas vendndodhjes (Prishtinë, Tiranë)
- **Hyrja** - Login me validim për email dhe fjalëkalim

## Teknologjitë

- HTML5
- CSS3 (Glassmorphism, animacione)
- JavaScript (Vanilla)

## Si të Fillosh

```bash
git clone https://github.com/JonKovaqi/mySpot.git
cd mySpot
```

Hap `index.html` në shfletues.

## Struktura

```
mySpot/
├── index.html           # Faqja kryesore
├── pages/               # Faqet tjera
├── scripts/             # JavaScript
├── styles/              # CSS
└── assets/              # Imazhe & ikona
```
